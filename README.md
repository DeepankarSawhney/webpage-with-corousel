# Camera Advisor

This repo is home to **Camera Advisor**, an Android app for the Galaxy S26 Ultra that watches the
live viewfinder and gives real-time suggestions for what to tweak (ISO, shutter speed, focus,
white balance, exposure compensation) to get a well-exposed, sharp, well-balanced shot.

The app lives in [`android-camera-advisor/`](android-camera-advisor/) — see that directory's
[README](android-camera-advisor/README.md) for how it works, how to build it, and how to verify it
on a device.

## Legacy content

`index.html`, `task.css`, and `slick/` at the repo root are an older static HTML/jQuery image
carousel demo, unrelated to the camera app, kept here for reference.
